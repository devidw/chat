## Keyboard Shortcuts

| Shortcut | Action                 |
| -------- | ---------------------- |
| ⌘ T      | Open Chat Browser      |
| ⌘ W      | Close Current Chat Tab |

### In Chat Browser

| Shortcut | Action                         |
| -------- | ------------------------------ |
| ↑ ↓ → ←  | Navigate                       |
| ⌘ A      | Add New Chat                   |
| ⌘ D, ↵   | Delete Chat (Enter to Confirm) |
| ⌘ R, ↵   | Rename Chat (Enter to Confirm) |
| ↵        | Open Selected Chat             |
